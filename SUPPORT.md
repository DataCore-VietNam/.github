# Support

Thanks for using DataCore VietNam software. Here is how to get help.

## Questions and usage help

- Check the repository's `README.md` and documentation first.
- Search existing [issues](https://github.com/orgs/DataCore-VietNam/repositories)
  to see if your question has already been answered.
- Open a new issue using the **Question** template if you cannot find an answer.

## Bug reports and feature requests

Use the issue templates provided in each repository:

- **Bug report** — for unexpected or broken behavior
- **Feature request** — for proposing new functionality

## Commercial and data inquiries

For questions about datasets, data delivery, or commercial use, contact us at
**contact@datacore.vn** or via [datacore.vn](https://datacore.vn).

## Security issues

Do not report security vulnerabilities through public issues. See our
[Security Policy](SECURITY.md) instead.

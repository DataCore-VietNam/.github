# DataCore VietNam

**Empowering Data-Driven Decisions Across Vietnam and Beyond**

DataCore is a trusted data platform for academic, corporate, and government
research. We provide comprehensive datasets, advanced analytics tooling,
real-time data validation, and flexible delivery options so teams can make
confident, evidence-based decisions.

## What we build

- **Datasets** — curated, validated, and continuously maintained data products
- **Analytics tooling** — libraries and services for exploring and modeling data
- **Data validation** — real-time quality checks and reconciliation pipelines
- **Delivery** — APIs, exports, and integrations that fit existing workflows

## Tech stack

Our repositories are primarily built with **Python**, **TypeScript / Node.js**,
and **Docker**. Shared CI workflow templates for these stacks are available to
every repository in this organization.
